# 🌦️ Weather App

A responsive Weather Application built using *HTML, CSS, and JavaScript, integrating the **Open-Meteo API* to provide real-time weather information based on user input.

## 🚀 Features

- Real-time weather data fetching using Open-Meteo API
- Search weather information by city/location
- Responsive design for desktop, tablet, and mobile devices
- Form validation for user inputs
- Error handling for invalid locations and API failures
- Clean and intuitive user interface
- Asynchronous data fetching using Fetch API and Async/Await

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6+)
- Open-Meteo API
- Git & GitHub

## 📌 Key Learnings

- API Integration
- Asynchronous JavaScript Programming
- DOM Manipulation
- Form Validation
- Error Handling
- Responsive Web Design
- Version Control with Git

## ⚙️ How It Works

1. Enter a city/location name.
2. The application validates the input.
3. Weather data is fetched from the Open-Meteo API.
4. Current weather details are displayed instantly.
5. Appropriate error messages are shown when needed.
